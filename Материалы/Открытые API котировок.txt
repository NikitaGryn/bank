MOEX ISS API: https://iss.moex.com/iss/reference/ — котировки, свечи, сделки, инструменты и исторические данные Московской биржи; задержанные данные доступны бесплатно.

Alpha Vantage: https://www.alphavantage.co/documentation/ — котировки и история акций, валют, криптовалют и сырья, а также технические индикаторы; нужен API-ключ, есть бесплатный тариф с лимитами.

Finnhub: https://finnhub.io/docs/api — текущие и исторические котировки акций, валют и криптовалют, новости и фундаментальные данные; нужен API-ключ.

Twelve Data: https://twelvedata.com/docs — котировки, свечи и технические индикаторы для акций, ETF, валют и криптовалют; есть бесплатный тариф с ограничением запросов.

Marketstack: https://marketstack.com/documentation — текущие и исторические биржевые данные по акциям разных стран; нужен API-ключ, бесплатный тариф ограничен.

Polygon.io: https://polygon.io/docs — котировки, сделки, агрегированные свечи и справочная информация по акциям, опционам, валютам и криптовалютам; часть данных доступна только на платных тарифах.

Financial Modeling Prep: https://site.financialmodelingprep.com/developer/docs — котировки, исторические цены, финансовая отчетность, дивиденды и профили компаний; нужен API-ключ.

EODHD: https://eodhd.com/financial-apis/ — биржевые котировки, исторические данные, фундаментальные показатели, дивиденды и корпоративные события; доступ зависит от тарифа.

CoinGecko API: https://docs.coingecko.com/reference/introduction — текущие и исторические котировки криптовалют, данные о рынках, биржах и торговых объемах; есть бесплатный демонстрационный тариф.

MOEX ISS API
Нужно для получения котировок российских акций, облигаций, валют и индексов Московской биржи.

Как использовать: приложение отправляет запрос с тикером, например SBER, и получает цену, объём торгов или свечи. Подходит для российского брокерского приложения лучше всего.

Alpha Vantage
Предоставляет данные по зарубежным акциям, валютам, криптовалютам и техническим индикаторам.

Как использовать: зарегистрироваться, получить API-ключ и отправлять запросы с тикером, например AAPL. Подходит для графиков и аналитики.

Finnhub
Даёт котировки, новости, финансовые показатели компаний и информацию о событиях.

Как использовать: получить API-ключ и запрашивать данные по тикеру. Можно использовать для карточки компании, новостей и инвестиционной аналитики.
